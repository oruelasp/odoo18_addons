Warehouse Management

Warehouse Management

This module enables you to control/restrict user access to:

  1. Specific warehouses
  2. Specific locations
  3. Operation types

Description

Description

This module offers comprehensive control over user access privileges within your organization's warehouse management system. With this module, administrators can precisely define which warehouses, storage locations, and operation types each user or user group can access.
By tailoring access permissions to specific roles and responsibilities, this module enhances data security, minimizes errors, and streamlines warehouse operations. It empowers businesses to maintain strict control over their inventory management processes while ensuring compliance with organizational policies and operational efficiency.

Standout Features

Standout Features

  1. Simple & easy Configuration.
  2. Works with both Enterprise and Community editions of Odoo.
  3. Control Multi Warehouse in single Company.
  4. Control multiple location under single warehouse.
  5. Control Access to Receipt and Deliveries.

Exploring the module: (How to)

Exploring the module: (How to)

  • Download and Install the module.
  • In the settings app select Restrict Stock Warehouse Location & Operations
  • Go to Preferences & select the user whom you want to give permission to specific operation type, Location and/or Warehouse
  • Restricted user will have accessed to assigned operation, location and/or warehouse only. If warehouse is selected here user can access all operation types.
  • We can also only assign specific operation type to user
  • Restricted User will only be able to see receipts now.
  • Restricted User will only be able to see receipts in operation type.
